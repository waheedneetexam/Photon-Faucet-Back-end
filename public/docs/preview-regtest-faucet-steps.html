Preview Integration Guide

Bitcoin RegTest Faucet Steps

Use these steps from photonbolt.xyz/preview to fund Photon Wallet with RegTest BTC from the live PhotonBolt faucet at faucet.photonbolt.xyz.

1. Connect Photon Wallet on the preview page

Use the Connect Photon Wallet button in the preview header. This exposes the currently active Photon wallet address to the page so you can copy the right RegTest address.

2. Copy the RegTest address

From the preview faucet section, copy the connected wallet address. This must be the same wallet you plan to use for UTXO creation, RGB issuance, and Lightning testing.

3. Open the live RegTest faucet

Open https://faucet.photonbolt.xyz/ and paste the copied address into the Bitcoin faucet form.

4. Request BTC and return to the wallet

Submit the request, then return to Photon Wallet. Refresh the balance or reopen the wallet popup so the new RegTest BTC appears.

5. Use the funded wallet for RGB actions

After BTC arrives you can create RGB UTXOs, issue assets, open invoices, and prepare for RGB Lightning channel flows.

  • RGB issuance requires BTC and confirmed UTXOs.
  • First-time RGB flows often need a prepared RGB UTXO before invoice creation or receiving works smoothly.
